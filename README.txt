Personal Assistant Software Program
The main objective of the program we developed is to help users streamlining tasks, prioritise goals, and improve productivity. 
This program contains 3 applications namely To-do list manager, Timer App, and Flashcard Quizzer.

Brief Introduction
- To-Do List Manager is a task management application designed to help users organize and track their daily activities. It provides essential task management features with a user-friendly interface for efficient personal organization.
- Timer App is made to assist people with various purposes to chronologically keep track of their endeavors like tasks and activities. A user of this Timer App can utilize various functionalities included in the Timer App.
- Flashcard Quizzer is an application designed to help users practice and test their knowledge on a variety of topics using flashcards. The app supports a number of features that are applicable to create, edit, delete quizzes. Users can also keep track of their progress on attempting quizzes, providing valuable insights into their learning journey.

Installation
1. Ensure you have Python 3.x installed on your system
Check version: python --version
If Python is not installed:
Download Python from python.org
During installation, make sure to check "Add Python to PATH"

2. Install Visual Studio Code (VS Code)
Download VS Code:
Go to code.visualstudio.com
Click the download button for your operating system
Run the installer

3. Install Python Extension in VS Code:
Open VS Code
Click the Extensions icon on the left sidebar (or press Ctrl+Shift+X)
Search for "Python"
Install the Python extension by Microsoft

4. Run the Program
Open the terminal in VS Code
Click the Run icon on the upside right


----------------------------
If you encounter any issues:

Make sure Python is correctly added to PATH
Try restarting VS Code after installation
Check if all required files are in the correct folder
